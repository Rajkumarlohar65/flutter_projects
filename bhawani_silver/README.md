## Features

- Light/dark mode toggle
- Email Authentication
- Live previews
- Fullscreen mode
- Cross platform


## Tech Stack

**Application:** Flutter (Cross Platform)

**Database:** Cloud Firestore

**Authentication:** Firebase Email Authentication

**Running in Web**
